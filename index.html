EV Adoption Forecaster for Washington State

A Streamlit dashboard that provides a 3-year forecast of electric vehicle adoption for specific counties in Washington State using time-series analysis (ARIMA).

Role: Data Analyst & ML Developer

Dashboard Landing

County Selection Dropdown

Cumulative Forecast Graph

Problem

Government agencies and charging infrastructure providers lack reliable, location-specific forecasts for EV growth, hindering effective planning and investment.

Approach

Used time-series data and ARIMA models to predict cumulative EV counts for 36 months, wrapped the model in a user-friendly Streamlit application for public consumption.

Outcome

Delivered forecasts with low error margins, providing a crucial tool for regional energy providers and urban planners to anticipate demand and strategize infrastructure deployment.

Key Highlights

3-Year ARIMA Time-Series Forecasting Interactive Streamlit Dashboard County-level Custom Predictions

Technology Stack

Python ARIMA Pandas Streamlit
